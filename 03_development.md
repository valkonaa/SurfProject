# Этап 3: Разработка

**Цель:** Реализовать MVP на Python с использованием FastAPI.

**Реализовано 3 фичи:**
1. `GET /workshops` — просмотр каталога.
2. `POST /bookings` — создание бронирования.
3. `GET /admin/bookings` — административная панель.

Код размещен в файле `main.py`. Запуск производится через `uvicorn main.py:app --reload`.
